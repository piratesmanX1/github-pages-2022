git status // check the status of the local workfile compare to the repository's
git add . // add as in adding all the files into commit ; '.'' stands for all
git commit -m initialcommit // commit your local working files changes into Github
git push origin main // push the committed files into the selected branch, in this case it will be 'main' branch
git checkout -b styling // creating a new branch name 'styling' via 'git checkout -b'
  • To initialize the Github Pages feature, go to GitHub main page and navigate to the repository, click on the Setting tab and scroll down to the GitHub section.
  • Select the source, navigate on the repository's branch (such as 'main') that you wish to initialize the GitHub Page.
  • Keep the second option as 'root', then save.
